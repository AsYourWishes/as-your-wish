# IDEA 快捷键

| 快捷键             | 功能                            |
| ------------------ | ------------------------------- |
| ctrl + alt + space | 补全提示                        |
| alt + enter        | 错误提示                        |
| ctrl + alt + T     | 代码块提示（if、try / catch等） |
| alt + insert       | 快速生成（重写、实现等）        |
| ctrl + J           | 常用代码                        |
| ctrl + alt + M     | 将代码抽取为方法                |
| alt + shift        | 移动单行代码                    |
| ctrl + shift       | 移动代码块                      |
| ctrl + D           | 复制一行                        |
| ctrl + alt + L     | 格式化代码                      |
| ctrl + alt + V     | 补全声明变量                    |
